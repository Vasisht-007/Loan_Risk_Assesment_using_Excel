# Loan_Risk_Assesment_using_Excel
Professional loan risk analysis with interactive dashboard

## Overview

Analyzed **4,269 loan applications** to build a simple, working risk assessment model.

**3 factors. 1 framework. Real validation.**

## Key Findings

- **Credit Score** drives 36% of risk variance
- **Affordability** (Income vs Loan) significantly impacts approval
- **Collateral Coverage** provides protection against defaults
- Model predictions match actual lending decisions

## What's Inside

- `Loan_Risk.xlsx` - Interactive Excel dashboard with 6 charts and filters
- `dashboard_screenshot.png` - Dashboard visualization
- `PROJECT_SUMMARY.md` - Detailed methodology and findings

## The Model

```
Risk Score = (Credit Risk × 40%) + (Income Risk × 30%) + (Collateral Risk × 30%)
```

Simple framework. Proven results.

## Key Metrics

- Total Loans Analyzed: 4,269
- Approved: 2,656 (62.3%)
- Rejected: 1,613 (37.7%)
- Average Risk Score: 0.67/1.0
